# cat-fight-dog  
## English  
It's a small game,just shut the foolish dog and defeat it!  
### how to play
Press up,down,left and right to move  
Press Z to shot mouse  
Press X to set a ball  
Try to be close to the dog!  
## 简体中文  
这是一个小游戏，只需要射击这只笨狗打败它！
### 如何操作
按上下左右移动  
按Z键发射老鼠  
按X键放置一个球  
试着贴近这只狗吧！
